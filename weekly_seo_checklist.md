- [ ] Check for new queries driving traffic
- [ ] Review page performance
- [ ] Check for coverage errors
- [ ] Monitor Core Web Vitals

- [ ] Check organic traffic trends
- [ ] Review conversion rates

## 🚨 **Red Flags to Watch For**

### **Immediate Action Required:**
- [ ] Traffic drops >20% in 24 hours
- [ ] Major ranking losses (>10 positions)
- [ ] Technical errors (404s, 500s)
- [ ] Manual penalties from Google
- [ ] Security issues or malware

### **Weekly Review Required:**
- [ ] Performance changes >15%
- [ ] New competitor movements
- [ ] Content performance issues
- [ ] Conversion rate changes >10%
- [ ] Mobile usability problems

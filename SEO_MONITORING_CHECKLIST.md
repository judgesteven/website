## 🔧 **Common Issues & Quick Fixes**

### **Issue: Pages Not Indexed**
**Quick Fix:**
1. Go to GSC → URL Inspection
2. Enter the URL
3. Click "Request Indexing"
4. Wait 24-48 hours

### **Issue: Core Web Vitals Poor**
**Quick Fix:**
1. Run PageSpeed Insights
2. Optimize images (compress)
3. Minimize JavaScript
4. Enable browser caching

### **Issue: Mobile Usability Errors**
**Quick Fix:**
1. Check viewport meta tag
2. Ensure touch targets are large enough
3. Fix text that's too small
4. Remove horizontal scrolling

### **Issue: Duplicate Content**
**Quick Fix:**
1. Set canonical URLs
2. Use 301 redirects
3. Consolidate similar pages
4. Remove duplicate meta descriptions
